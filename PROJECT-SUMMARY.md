# Advanced Design Certificate in AI-Augmented Cultural Systems & Experiences

## Overview

This is a **nine-microcredential (MC) certificate program** — an educational curriculum designed to teach competencies in cultural production, institutional mediation, and AI-augmented systems design. It's structured as an Obsidian vault with Git version control.

**Version:** 0.2.0
**Status:** In Development

## What's Been Built So Far

- **MC1 (Production Literacy)** — Complete with audio essay and podcast analysis assignments
- **MC2 (Experience Design Awareness)** — Full syllabus + Ballroom Transformation assignment
- **MC3 (Platform & Ecosystem Economics)** — Full Cultural Product Tour Strategy syllabus
- **MC7 (AI-Augmented Systems Design)** — Two versions, grounded in Activity Theory and Change Laboratory methodology
- **MC4–MC6, MC8–MC9** — Still skeleton outlines (~1 KB each)

## Key Characteristics

- **Dialectical pedagogy** — thesis/antithesis/synthesis resolution between cultural producers and institutions
- **Theory-grounded** — Activity Theory, boundary objects, Change Laboratory methodology, Strategyzer tools
- **Live capstone (MC9)** — Students design and defend a real fundraising event
- **Modular/licensable** — Designed for adoption by multiple institutions (Carleton, BCIT, SFU SIAT, Emily Carr)
- **Real-world case study** — ClearLED B2B studio incubation at Multimodal Media Lab (Vancouver)

## Key Frameworks & Theoretical Grounding

- **Activity Theory** — modeling activity systems and structural contradictions
- **Change Laboratory methodology** — Strategyzer tools as second stimulus
- **Boundary object theory** — fundraiser event as shared artifact coordinating different institutional logics
- **Formative intervention** — design as systemic change
- **Strategyzer tools** — Business Model Canvas, Value Proposition Canvas
- **Claude Skills** — AI infrastructure for workflow design and practice augmentation

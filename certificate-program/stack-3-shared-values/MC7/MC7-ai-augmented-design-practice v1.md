## Description

Introduces students to the process of founding and operating an AI-augmented design practice, using Strategyzer tools as the strategic framework and Claude Skills as the operational infrastructure layer. 

This microcredential attempts to resolve the tensions between cultural processes (Stack 1) and institutional resources (Stack 2) by equipping students with the practical tools to build sustainable creative practices that honour both cultural integrity and institutional requirements.

The ClearLED B2B studio incubation project — a bespoke branding and marketing studio being built from the ground up inside Multimodal Media Lab (Vancouver) — serves as the primary guided case study. Students also complete a self-directed case study of their own choosing.

---

## Learning Outcomes

- [ ] Apply Strategyzer frameworks (Business Model Canvas, Value Proposition Canvas) to design practice formation
- [ ] Use Claude Skills as an infrastructure layer for studio operations and talent compensation
- [ ] Develop a previsualisation workflow for marketing and branding using AI tools
- [ ] Analyze a guided case study (ClearLED/MML) and extract transferable principles
- [ ] Produce a self-directed case study applying the same methodology to a practice context of the student's choosing

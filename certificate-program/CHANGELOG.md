## [0.2.0] — 2026-02-19

- Certificate renamed to **Advanced Design Certificate in AI-Augmented Cultural Systems & Experiences**
- MC7 reframed and retitled from "AI-Augmented Design Practice" to **"AI-Augmented Systems Design"**
- MC7 grounded in Activity Theory, Change Laboratory methodology, and boundary object theory
- MC9 capstone fully articulated as a live formative intervention — fundraising event as boundary object
- Capstone deliverables defined: activity system modeling, Strategyzer ecosystem design, governance and IP structuring, written proposal, pitch deck, live stakeholder defense simulation, client/sponsor portfolio

- Program map updated: MC1–MC3 status changed to "Developed"; MC7 to "In Development"
- Emily Carr University added to institutional alignment table
- Stack 3 description updated to reflect systems design and live intervention framing
